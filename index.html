The Easy Save project is a project I did at school in a team of 4. It's a desktop application that manages backups. It is an application coded in C# and we used Azure Devops as a Git server to manage the versions of the project.

It's a project I'm particularly proud of, because I had never done desktop application development and we did very well.

This application could manage several dozen backups, run them at the same time, pause them, etc... There was even a feature to monitor and interact with the application with another computer on the network.

Easy Save
×

This is my very first corporate project where I was the main actor.

Xenocs commissioned me to build a chip programming station. These chips were then integrated into the samples which were going to be analyzed by the XEUSS (one of the machines manufactured by Xenocs and used for characterization). Depending on what was written in the chip (the size of the sample, its viscosity, etc.), the machine adopted a certain behavior.

So I made the station with the help of some colleagues that I asked, and I created a graphical interface to write more easily on the chips. The logic was in Python and the GUI was made with QT

It was an extremely rewarding project, knowing that even after my BTS passed, the chip programming station was still in use a year later and I continued to improve it.

Unfortunately I don't have an image of the project to show since it belongs to the company.
